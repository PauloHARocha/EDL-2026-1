# Lista de exercícios — Pilhas

1. O que é e para que serve uma pilha?

2. Em que situações uma pilha pode ser utilizada?

3. Dadas as operações de pilha insere I e remove R, escreva a configuração final da pilha após as seguintes operações: I(10), I(20), R, I(30), I(45), I(21), R, R, I(50), I(60), R, I(70), R.

4. Escreva um programa que utilize uma pilha para verificar se expressões aritméticas estão com a parentização correta. O programa deve verificar expressões para ver se cada "abre parênteses" tem um "fecha parênteses" correspondente. Por exemplo:

    Correto:  
        - `( ( ) )`
        - `( ( )( ) )`
        – `( ) ( )`

    Incorreto:  
        - `)(` 
        – `( ( ) (` 
        – `) ) ( (`

5. Desenvolva uma função para inverter a posição dos elementos de uma pilha P.

6. Considere uma pilha que armazene caracteres. Escreva uma função que verifique se uma palavra é um palíndromo.
